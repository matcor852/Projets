Projet client/serveur de jeu de soirée style action ou vérité en réseau.
Nécessite un client par joueur et repose sur un système de parties de 2 joueurs minimum et 20 joueurs maximum.

Les joueurs choisissent un icon qui sera visible par les autres joueurs. Les choix se font aléatoirement, que ce soit le choix du joueur, le fait que 
ce soit une action ou une vérité, ainsi que l'action ou la vérité choisie dans la base de données.
La base de données est remplie et modifiée depuis des forums et par les joueurs sur demande.
